![git-commit.png](http://193.112.98.8/atomImg/git/git-commit.png)

### 将本地库 提交到远程库 

> git push

![git-push.png](http://193.112.98.8/atomImg/git/git-push.png)



快捷键 

```
ctrl + shift + k
```

或

```
alt + 1 + 8
```
